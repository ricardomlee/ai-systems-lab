# Notes

This folder is for daily notes, ideas, and random thoughts.

## How to Use

- Create a new file for each note: `YYYY-MM-DD-note-title.md`
- Use for brainstorming, meeting notes, random ideas
- Don't overthink - just write

## Templates

### Note Template

```markdown
# Note Title

**Date**: YYYY-MM-DD
**Related**: [[link to other notes]]

## Context

Why am I writing this?

## Main Content

Whatever you're thinking about...

## Action Items

- [ ] Things to do later
```
