# Project Ideas

Potential projects to build and showcase expertise.

---

## Project 1: Rust LLM Inference Engine

**Goal**: Build a minimal but fast LLM inference engine

**Tech Stack**: Rust, CUDA (optional), GGUF format

**Features**:
- Load GGUF models
- KV cache for efficient generation
- Continuous batching
- OpenAI-compatible API

**Why**: Shows deep understanding of LLM internals

**Timeline**: 4-6 weeks

---

## Project 2: Vector Search Library

**Goal**: Implement HNSW-based vector search in Rust

**Tech Stack**: Rust, memory-mapped files

**Features**:
- HNSW index construction
- Efficient ANN search
- Persistence layer
- Python bindings via PyO3

**Why**: RAG infrastructure is hot, shows systems + AI skills

**Timeline**: 3-4 weeks

---

## Project 3: Edge AI Toolkit

**Goal**: Collection of tools for edge AI deployment

**Tech Stack**: C/Rust, cross-compilation

**Features**:
- Model quantization utilities
- Memory profiling tools
- Benchmark suite
- Deployment scripts

**Why**: Positions you as edge AI expert

**Timeline**: 4-6 weeks
